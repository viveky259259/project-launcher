Everything you need, nothing you don't

Built for developers who juggle multiple projects across multiple stacks.

⚡

Instant Launch

Open any project in Terminal, VS Code, or Finder with one click. No more cd ~/Projects/that-thing.

💚

Health Scores

Every project gets a 0-100 health score based on git activity, dependency freshness, and test coverage.

🔍

Smart Filters

Filter by activity (this week, last month), health status, git state, language, or custom tags.

📊

Code Wrapped

Beautiful year-in-review stats with shareable cards. Your commits, streak, top language — all branded and ready to share.

🚀

Rust-Powered

Native Rust FFI core using libgit2. Git operations run at native speed without spawning shell processes.

🛠

Tech Stack Detection

Auto-detects 17+ languages and frameworks. Handles multi-tech projects like Flutter+Rust or React+Go monorepos.

📦

Export & Git Sync

Zip and push all your projects to GitHub with one click. Smart sync skips already-uploaded projects. Auto-splits large files.

🔒

Token-Based Auth

Push to any GitHub repo using personal access tokens. Tokens are never stored or shown in error messages.

🚀

Ship Readiness

Checklist with auto-detect, manual, and AI-assisted checks. Smart release process detection follows your project's own workflow.

📜

Release Management

Version tracking, release pulse, one-click ship, and daily release habits — all from the status bar.

🛡

Enterprise Compliance

License detection, secret scanning, SBOM generation, and code signing checks. Compliance audit in seconds.

⌨

CLI Tool

Use plauncher from the terminal — list, search, open, and check health across all your projects.

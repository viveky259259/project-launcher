How it works

Three steps to clarity

From scattered folders to a single source of truth.

1

Scan your machine

Point it at your projects folder. It auto-discovers every git repo, detects the tech stack, and builds your dashboard.

2

See what matters

Health scores, staleness indicators, unpushed commits, and activity timelines — all at a glance.

3

Launch and ship

One click to open in your editor. Ship readiness checks, release management, and export — right from the dashboard.

Features

Everything you need, nothing you don't

Built for developers who juggle multiple projects across multiple stacks.

⚡

Instant Launch

Open any project in Terminal, VS Code, or Finder with one click. No more cd ~/Projects/that-thing.

💚

Health Scores

Every project gets a 0-100 health score based on git activity, dependency freshness, and test coverage.

🔍

Smart Filters

Filter by activity, health status, git state, language, or custom tags. Find anything in seconds.

📊

Code Wrapped

Beautiful year-in-review stats with shareable cards. Your commits, streak, top language — ready to share.

🚀

Rust-Powered

Native Rust FFI core using libgit2. Git operations run at native speed without spawning shell processes.

🛠

Tech Stack Detection

Auto-detects 17+ languages and frameworks. Handles multi-tech projects like Flutter+Rust monorepos.

📦

Export & Git Sync

Zip and push projects to GitHub with one click. Smart sync skips already-uploaded projects.

📜

Release Management

Version tracking, release pulse, one-click ship, and daily release habits — all from the status bar.

🚀

Ship Readiness

Checklist with auto-detect, manual, and AI-assisted checks. Smart release detection follows your workflow.

🔒

Token-Based Auth

Push to any GitHub repo using personal access tokens. Tokens are never stored or shown in error messages.

🛡

Enterprise Compliance

License detection, secret scanning, SBOM generation, and code signing checks. Audit in seconds.

⌨

CLI Tool

Use plauncher from the terminal — list, search, open, and check health across all your projects.
